# 应用密码快速访问指南

## 🎯 最快方法：直接访问

**直接点击或复制以下链接**（需要先登录 Google 账户）：

```
https://myaccount.google.com/apppasswords
```

这个链接会直接带您到应用密码页面。

## 📍 详细步骤

### 步骤 1: 确认两步验证已启用

从您的截图可以看到，两步验证已经启用（显示"启用时间: 21:24"），所以可以直接进入下一步。

### 步骤 2: 进入应用密码页面

**方法 1: 直接访问链接**（推荐）
- 访问：https://myaccount.google.com/apppasswords

**方法 2: 通过安全设置**
1. 在当前页面，点击"两步验证"选项（不是复选框，是整个选项区域）
2. 进入两步验证详细页面
3. 向下滚动，找到"应用密码"部分
4. 点击"应用密码"

### 步骤 3: 生成应用密码

1. 选择应用：**邮件**（Mail）
2. 选择设备：**其他**（或输入"联系表单"）
3. 点击"生成"
4. **立即复制**生成的 16 位密码

## ⚠️ 重要提示

- 应用密码格式：`xxxx xxxx xxxx xxxx`
- **使用时要去掉空格**：`xxxxxxxxxxxxxxxx`
- 应用密码只显示一次，请立即保存！

## 🔗 相关链接

- **应用密码页面**：https://myaccount.google.com/apppasswords
- **两步验证设置**：https://myaccount.google.com/signinoptions/two-step-verification
- **安全设置主页**：https://myaccount.google.com/security
